# Word to Moodle
Script Convert Word to Moodle

## 1. Contoh Format DOCX

### Format Soal Standard (Single Answer)
```
1. Siapa presiden pertama Indonesia?
A. Soekarno
B. Soeharto
C. Habibie
D. Megawati
E. SBY
ANS: A

2. Ibukota Indonesia adalah...
A. Bandung
B. Jakarta
C. Surabaya
D. Medan
E. Makassar
ANS: B
```

### Format Multiple Answers
```
3. Yang termasuk negara ASEAN adalah...
A. Indonesia
B. Australia
C. Malaysia
D. Jepang
E. Singapura
ANS: A/C/E

4. Bilangan prima adalah...
A. 2
B. 4
C. 5
D. 6
E. 7
ANS: A/C/E
```


### Format dengan Matematika
Script otomatis mengkonversi:
- Equation Editor Word → LaTeX
- Pecahan campuran: `4²⁄₅` → `4\frac{2}{5}`
- Superscript/Subscript otomatis
- Gunakan Equation di word dengan cara ```Alt + =```
```
5. Hasil dari 4½ + 2¾ adalah...
A. 6¼
B. 7¼
C. 8¼
D. 9¼
E. 10¼
ANS: B

6. Nilai dari x² + 2x + 1 untuk x = 3 adalah...
A. 12
B. 14
C. 16
D. 18
E. 20
ANS: C
```
